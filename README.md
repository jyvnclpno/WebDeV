# WebDeV

A property management platform prototype for Philippine tenants and landlords.  
This project showcases landing pages, dashboards, and onboarding flows designed for usability and localization.

## Live Demo
[View WebDeV on GitHub Pages](https://jyvnc1pno.github.io/WebDeV/)

## Features
- Landing page with onboarding flow
- Tenant and landlord dashboards
- Login and sign‑in pages
- Localized currency (PHP) and payment options
